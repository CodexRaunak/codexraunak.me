## What I've been doing

  Mentoring work on a pipeline that automatically publishes plugin-modernization reports
  and visualizes them through interactive dashboards.
- **LFX Mentee — Meshery (CNCF).** Migrated `docs.meshery.io` from Jekyll to Hugo —

  automates plugin modernization: automated metadata extraction and a set of declarative
  and imperative OpenRewrite recipes. Authored a work-product blog on `jenkins.io`.
